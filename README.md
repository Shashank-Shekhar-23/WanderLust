# 🏡 Airbnb Clone

A full-stack Airbnb-inspired web application that allows users to discover, list, and book unique accommodations. This project replicates the core functionality of Airbnb, providing a seamless booking experience with modern UI/UX and scalable architecture.

## ✨ Features

### 🔐 Authentication & Authorization
- User registration and login
- Secure authentication
- Role-based access control
- Profile management

### 🏠 Property Listings
- Create, edit, and delete listings
- Upload property images
- Add amenities, pricing, and descriptions
- Manage listing availability

### 🔍 Search & Discovery
- Search properties by location
- Filter by price, amenities, and property type
- Responsive search experience
- Interactive property browsing

### 📅 Booking System
- Real-time availability checking
- Reservation management
- Booking history
- Check-in and check-out date selection

### 💳 Payments
- Secure payment integration
- Booking confirmation
- Transaction history

### 📱 Responsive Design
- Mobile-friendly interface
- Tablet and desktop support
- Modern and intuitive UI
